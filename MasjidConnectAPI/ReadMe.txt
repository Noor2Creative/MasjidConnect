MasjidConnect 
--------------
An **ASP.NET Core API for managing **Mosque information** and **Namaz (Salah) timings**.

Tech Stack
- **Backend:** ASP.NET Core  
- **Database:** MS SQL Server  
- **Authentication:** JWT (JSON Web Token)  

*******************************
Features:
*******************************
1. User and Admin Sign up/Login
2. Register/Delete/Update Masjid and it Namaz timings
3. Guest see Masjid and namaz timings
4. Authentication/Authorization using JWT
